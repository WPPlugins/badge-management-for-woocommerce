= Overview =

Customers are naturally drawn to see highlighted information when visiting your site. Badge Management Plugin allows you to highlight 'special' features and 'new' offers on your products, so as to catch your customers' attention. With this plugin you could customize a badge's text & its color and could also set its dimension and position. The plugin provides you with some default image-badges as well.

= Free features =

* Option to have Text Badge or Image Badge
* 4 Default Image-Badge options available to choose from 
* Option to customize text badges
* Option to Set Text Options ( Color, Font Size and Line Height)
* Option to Set Badge Position
* Option to Set Badge Dimensions 
* Option to Show or Hide 'On Sale' Badge, as per your choice

= Premium Features: =


* Option to Add a particular Badge to recent products.
* Option to Add a particular Badge to ‘On Sale’ products.
* Option to Add a specific Badge to featured products.
* Option to Hide the default WooCommerce ‘On Sale’ Badge.
* Option to Hide Badge in single Product Page.
* Option to Add a particular Badge to each Product Category.
* 9 Badge images available.
* Option to Upload your own images for Badges.
* Option to Set Badge Opacity.
* Option to Set Badge Position.
* Options (Border Radius, Padding, Opacity) for defining Badge Container.
* Option to Set Expiry Date for Badges.
* Option to Set Expiry Date & Time on any of the badges.

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
3. For the plugin to work as it should, WooCommerce plugin has to be installed and enabled.
